#### **Cheat Sheet \& Summary**



###### **Shortcuts**

* Win + R opens the Run dialog.
* Ctrl + Shift + Esc opens Task Manager.
* Win + X opens Quick Menu (Device Manager, Event Viewer).
* services.msc opens the Services console.
* eventvwr.msc opens Event Viewer.



###### **Services**

* Key services: Print Spooler, Windows Audio, DHCP Client.
* Startup types: Automatic starts at boot, Manual starts on demand, Disabled never starts.
* Always check Dependencies before stopping critical services.



###### **Event Viewer**

* Logs include Application, System, and Security.
* Filter by Level: Error or Critical.
* Export logs for escalation as .evtx.
* File naming: User\_Issue.evtx.





##### **Summary of Key Takeaways**

1. Always replicate issues before escalating.
2. Task Manager, Device Manager, Services, and Event Viewer are core Windows tools.
3. Document actions clearly in tickets.
4. Cheat sheet shortcuts and phrases improve efficiency and credibility.
5. Skills learned today are the foundation of every First-Line Support ticket.
6. Prioritize tickets by impact, know escalation paths, and use remote tools when necessary.
