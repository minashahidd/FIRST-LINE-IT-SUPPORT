#### **Labs \& Hands-On Exercises**



###### **Lab 1 — User Accounts**

Create 'mina' with Administrator rights

Create 'other' with Standard rights

Attempt software installation as 'other', blocked

Attempt system setting changes as 'other', blocked

Adjust admin privileges for 'other'

Test actions 'other' can perform

Note tasks that require admin rights



###### **Lab 2 — Task Manager \& System Performance**

Open Task Manager with Ctrl + Shift + Esc

Monitor CPU, RAM, Disk, and Network usage while running applications

Add heavy applications to startup

Reboot and check performance

Disable unnecessary startup items

Reboot and compare performance

Note improvements and bottlenecks



###### **Lab 3 — Device Manager \& Drivers**

Open Device Manager with Win + X

Focus on network adapter and audio device

Disable and re-enable devices

Update drivers as needed

Check effect on device functionality



###### **Lab 4 — Services**

Open Services with Win + R → services.msc

Stop, start, and restart Print Spooler, Windows Audio, DHCP Client

Change Startup type: Automatic, Manual, Disabled

Check Dependencies for each service

Note impact of stopping services on dependent functions



###### **Lab 5 — Event Viewer**

Open Event Viewer with Win + R → eventvwr.msc

Explore Application, System, and Security logs

Filter logs by Error and Critical levels

Simulate failed login attempt

Force Notepad to crash and record event

Stop Print Spooler service to generate log

Export logs as .evtx for documentation and escalation

Record key events for troubleshooting
