#### **Example Ticket Scenarios**



###### **Ticket 1 — User Cannot Log In**

Issue: Account locked due to failed login attempts.

Steps Taken: Verified credentials and unlocked the account.

Resolution: Advised password reset and confirmed access restored.



###### **Ticket 2 — Application Crashes**

Issue: Notepad crashes on launch.

Steps Taken: Checked Event Viewer and restarted application.

Resolution: Issue resolved and user confirmed.



###### **Ticket 3 — Printer Not Working**

Issue: Printing failed.

Steps Taken: Checked Services. Print Spooler was stopped, restarted the service.

Resolution: Printing restored.



###### **Ticket 4 — Slow PC**

Issue: User reports computer is very slow.

Steps Taken: Checked Task Manager. High CPU usage from startup apps. Disabled unnecessary apps.

Resolution: Performance improved and user confirmed.



###### **Ticket 5 — No Sound**

Issue: User cannot hear audio.

Steps Taken: Checked Windows Audio service. Stopped and restarted the service.

Resolution: Sound restored.
