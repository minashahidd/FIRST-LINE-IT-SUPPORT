##### **Ticket 1 – User Cannot Access Folder**



###### **Issue:**

###### **User cannot open C:\\SharedDocs**



**Actions:**

\- Verified account enabled and password valid

\- Checked group membership affecting access

\- Reviewed folder permissions

\- Adjusted permissions as necessary

\- Tested access to confirm resolution





##### **Ticket 2 – User Cannot Connect to Internet**



###### **Issue:**

###### **User unable to access websites**



**Actions:**

\- Pinged google.com and local gateway

\- Checked IP configuration (IP, subnet, gateway, DNS)

\- Verified network adapter is enabled

\- Retested connectivity





##### **Ticket 3 – User Reports Slow PC**



###### **Issue:**

###### **User reports slow performance**



**Actions:**

\- Opened Task Manager → monitored CPU, memory, disk usage

\- Identified high-resource processes

\- Checked Windows services

\- Reviewed Event Viewer for errors or warnings

\- Documented troubleshooting steps
