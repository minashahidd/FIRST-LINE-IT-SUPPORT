###### **Network Commands:**

* ping \[host] -n 3 → connectivity
* tracert \[host] → trace route
* ipconfig /all → IP, gateway, DNS



###### **Local Users \& Groups:**

* lusrmgr.msc → manage accounts and groups
* Verify account enabled, password valid
* Check group membership
* Reset password or create test users



###### **Permissions \& Access:**

* Read / Write / Modify / Full Control
* Explicit vs Inherited
* Workflow: identify → verify → adjust → test → document



###### **Core IT Tools:**

* Task Manager → monitor processes and performance
* Services.msc → check/manage Windows services
* Event Viewer → logs
* File Explorer Security → folder permissions



###### **Shortcuts:**

* Win + R → Run dialog
* Ctrl + Shift + Esc → Task Manager
* Win + Pause/Break → System properties
* Shift + Right-Click → Run as different user
* Win + R → eventvwr.msc
* Win + R → services.msc
